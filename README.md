# Frontend-Dev-Tools

## What's in the box?
 - Express web server, open browser to http://localhost:3000/
 - ECMA script: Babel-Core/Browserify/Babelify (supports es2015+)
 - Node-Sass
 - Style linter
 - ECMA linter

## Setup (It's fast, fun and easy!)
> Install packages:
```
$ npm install
```
> To run dev environment:
```
$ npm start  
```
> To run production build:
```
$ npm run build  
```


## Utilities
> ECMA Script Linter:
```
$ npm run es-lint
```
> Style Linter:
```
$ npm run style-lint
```
